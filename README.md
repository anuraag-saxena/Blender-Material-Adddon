* Installing the addon
    1. Download the zip folder
    2. Extract and keep only the ".py" extension file
    3. Open Blender, <br>
    Edit > Preferences > Add-Ons
    4. Choose Install button and select the MaterialsAddOn.py file.
    5. The addon is installed.
